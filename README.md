# Connect 4- peli
Pygamella toteutettu peli, jossa joko 2 pelaajaa tai 1 pelaaja ja tietokone kilpailee siitä kumpi saa 7x6 ruudukkoon oman värisiä nappuloita 4 vierekkäin joko pystyyn, vaakaan tai viistoon.
