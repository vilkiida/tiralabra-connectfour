# Connect 4- peli
Pygamella toteutettu peli, jossa 2 pelaajaa kilpailee siitä kumpi saa 7x6 ruudukkoon oman värisiä nappuloita 4 vierekkäin joko pystyyn, vaakaan tai viistoon.
